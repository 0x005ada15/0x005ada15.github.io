## 名前付き新規セッションを開始
```sh
tmux new -s session-name
```

## セッションを終了 ※-t <対象セッション名>でセッション名の指定も可能
```sh
tmux kill-session -t session-name
```

## セッションを終了 ※-t
```sh
tmux kill-session 
```

## tmux全体を終了
```sh
tmux kill-server
```
